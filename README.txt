BESCHREIBUNG/FEATURES
- Spiel wird mit Space gestartet
- Das Anwendungsfenster kann mit Escape geschlossen werden
- grundlegendes Gameplay (Ball zerstört Blöcke, Paddle kann bewegt werden, Ball kann "sterben", wenn er Paddle verfehlt)
- Paddle-Item: Paddle wird breiter
- Ball-Item: zwei zusätzliche Bälle werden gespawnt; ziehen keine Leben ab, wenn sie "sterben"
- Punktezähler (grüne Blöcke: 100, blaue Blöcke: 200, gelbe Blöcke: 300, rote Blöcke 400)
- Blöcke müssen verschieden oft getroffen werden, um zerstört werden zu können (grün: einmal, blau: zweimal, gelb: dreimal, rot: viermal)
- 3 verfügbare Leben, wenn der weiße Ball "stirbt", wird ein Leben abgezogen und der Ball sowie das Paddle resettet
- Spiel gewonnen, wenn alle Blöcke zerstört wurden --> "GAME WON"-Nachricht erscheint
- Spiel verloren, wenn alle 3 Leben verbraucht --> "GAME OVER"-Nachricht erscheint
- Das Spiel resettet sich selbst, wenn man nachdem man verloren hat, eine neue Runde startet

QUELLEN
- Sound: 8BIT Retro Beep aus Vorlesung/Pong-Projekt
